# Django Blog ProjectThis is a simple blog web application built with Django.  It allows users to create, edit, and delete posts.## Features- User authentication (login/register/logout)- Create blog posts- Edit and delete posts- View list of posts- Responsive basic UI## Tech Stack- Python 3- Django- SQLite (default database)- HTML / CSS (basic templates)## Installation### 1.
